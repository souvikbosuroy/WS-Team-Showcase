## Description

**Souvik WS Team Showcase** is an enterprise-grade WordPress plugin designed specifically for **Elementor Pro**. It enables site administrators to create beautiful manual or dynamic team grid showcases with advanced custom features:

* 🧩 **ACF Database Integration:** Dynamically populates team member profiles directly from custom post types and custom fields, synchronized seamlessly into both WordPress and ACF dashboards.
* 🔍 **Coordinated Filters & Search:** Search and filter members dynamically with advanced responsive flex layout controls.
* 📌 **Sticky Filter Bar Offset:** Precise client-side scroll tracking ensures the sticky filter bar aligns perfectly below active headers or admin bars.
* ⚡ **Coordinated GSAP Animations:** Stunning scroll-triggered entry movements powered by GSAP conditional enqueuing.
* ♿ **Accessible Dialog Popups:** Fully accessible overlays with keyboard tab-containment, background scroll locks, and prefers-reduced-motion compatibility.
* 🎨 **True Vector brand social logos:** Standard SVG logos with robust CSS inheritance and brand color hover transitions.

---

## Installation

1. Upload the entire `souvik-ws-team-showcase` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Use the **Souvik WS Team Showcase** element in Elementor to add team showcases.

## Frequently Asked Questions

### Does it support ACF?
Yes, it automatically registers and synchronizes CPT (`team-member`) and custom taxonomy fields.

### Can I use custom brand icons?
Yes, administrators can paste alternative SVG vector tags directly in the Settings dashboard.
